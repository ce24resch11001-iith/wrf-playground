# FRC_URB2D: Global Urban Fraction Dataset for WRF

## Overview

**FRC_URB2D** represents the **urban fraction** within each WRF grid cell, defined as the fraction of the grid covered by urban or impervious surfaces.

This variable is commonly used by urban canopy parameterization schemes to improve the representation of urban land surfaces, energy balance, and urban-atmosphere interactions.

This repository documents the use of a high-resolution global urban fraction dataset derived from the **ESA WorldCover** land-cover product and prepared for direct integration with the WRF Preprocessing System (WPS).

---

## Dataset Description

The dataset was developed by **Patel and Roth (2022)** using the **ESA WorldCover 2020/2021** 10 m land-cover products.

Urban fraction values are generated by:

1. Identifying urban/built-up pixels from WorldCover.
2. Converting urban pixels to binary values (urban = 1, non-urban = 0).
3. Aggregating urban coverage to coarser spatial resolutions.
4. Computing the fraction of urban pixels within each output grid cell.

Urban fraction is calculated as:

[
\text{Urban Fraction} =
\frac{\text{Number of Urban Pixels}}
{\text{Total Number of Pixels}}
]

Values range from:

* **0.0** → No urban coverage
* **1.0** → Fully urbanized grid cell

| 0.009°     | ~1 km             |
| 0.0027°    | ~300 m            |
| 0.0009°    | ~100 m            |

The dataset is distributed in both:

* GeoTIFF format
* WRF-WPS geographical format

## WRF Integration

To use FRC_URB2D in WRF:

1. Download the required urban fraction dataset.
2. Extract the WPS files into the `WPS_GEOG` directory.
3. Add the following entry to `GEOGRID.TBL`:

```text
name=FRC_URB2D
        priority=1
        optional=yes
        dest_type=continuous
        fill_missing=0.
        interp_option=default:average_gcell(2.0)+four_pt
        rel_path=default:urb_fraction_1km/
        flag_in_output=FLAG_FRC_URB2D
```

4. Run `geogrid.exe`.
5. Verify `FRC_URB2D` in the generated `geo_em.d0*.nc` files.

## Urban Parameterization Applications

FRC_URB2D is used by urban canopy models to represent sub-grid urban heterogeneity and impervious surface coverage.

Common applications include:

* Single-Layer Urban Canopy Model (SLUCM)
* Building Effect Parameterization (BEP)
* BEP + Building Energy Model (BEM)

Urban fraction influences:

* Surface energy balance
* Heat storage
* Sensible and latent heat fluxes
* Surface temperature
* Urban heat island intensity

## Data Preparation Workflow

```text
ESA WorldCover (10 m)
          ↓
Identify Built-Up Pixels
          ↓
Binary Urban Mask
          ↓
Spatial Aggregation
          ↓
Urban Fraction Calculation
          ↓
GeoTIFF / WPS Files
          ↓
WPS geogrid.exe
          ↓
FRC_URB2D in geo_em.d0*.nc
```
